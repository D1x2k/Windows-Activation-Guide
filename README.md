# 🔑 Ручная Активация Windows в 3 команды (Guide by D1x2k)

> 📥 **Быстрый вариант:** Вы можете не вводить команды вручную, а просто скачать готовый скрипт [**Win.bat**](./Win.bat) из этого репозитория, запустить его от имени Администратора, и он сделает всё автоматически.

---

Если вы предпочитаете классический ручной метод, вы можете активировать систему самостоятельно. Для этого достаточно поочередно выполнить всего **три команды** в консоли.

### 🛠️ Инструкция по активации

1. Откройте **Командную строку** строго **от имени Администратора** (Нажмите `Win + X` -> Выберите *Командная строка (Администратор)* или *Терминал (Администратор)*).
2. Скопируйте **весь блок кода ниже целиком**, вставьте в консоль и нажмите **Enter**:

---

```cmd
:: 1. Установка публичного KMS-ключа (для Windows 10/11 Pro)
slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX

:: 2. Подключение к серверу лицензирования
slmgr /skms kms.digiboy.ir

:: 3. Запуск процесса активации в системе
slmgr /ato
