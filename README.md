# 🔑 Активация Windows (Guide by D1x2k)

### 📥Быстрый вариант
Вы можете не вводить команды вручную, а просто скачать готовый скрипт [**Windows Activation**](https://github.com/D1x2k/Windows-Activation-Guide/releases) из этого репозитория, запустить его от имени Администратора, и он сделает всё автоматически. Скрипт полностью безопасен, весит всего около 30 строк кода и не содержит скрытых загрузок или `.exe` файлов. Вы можете в любой момент открыть его через Блокнот, чтобы лично убедиться в чистоте кода и отсутствии вирусов.

---

Если вы предпочитаете классический ручной метод, вы можете активировать систему самостоятельно. Для этого достаточно поочередно выполнить всего **три команды** в консоли.

### 🛠️ Инструкция по активации

1. Откройте **Командную строку** строго **от имени Администратора** (Нажмите `Win + X` -> Выберите *Командная строка (Администратор)* или *Терминал (Администратор)*).
2. Скопируйте **весь блок кода ниже целиком**, вставьте в консоль и нажмите **Enter**:

---

1. Установка публичного KMS-ключа (для Windows 10/11 Pro)
```cmd
slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX
```
2. Подключение к серверу лицензирования
```cmd
slmgr /skms kms.digiboy.ir
```
3. Запуск процесса активации в системе
```cmd
slmgr /ato
```

> 💡 **Полезный совет:** Ключ в строке по умолчанию настроен на версию **Pro**. Если у вас установлена другая редакция, просто скопируйте нужный ключ из таблицы ниже и замените им первый блок символов (`W269N-...`) в команде перед тем, как нажать **Enter**.
